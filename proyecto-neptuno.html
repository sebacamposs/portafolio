Neptuno
Visor y editor de redes SATURN
Descripción

Neptuno es un visualizador y editor de redes de modelación SATURN. Cargas un archivo .dat, y este se despliega sobre un mapa base. Con el se pueden agregar y eliminar nodos, agregar y eliminar arcos, agregar y eliminar zonas, agregar pistas solo bus, entre otras opciones.

Tecnologías
Plataforma APIs REST Arquitectura modular Multiusuario
