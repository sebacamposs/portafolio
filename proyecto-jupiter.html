Jupiter
Generador de redes SATURN
Descripción

Jupiter es un generador de redes de modelación SATURN. Se define un polígono sobre el mapa, y se descarga información de Open Street Map, a partir de la cual se genera el archivo .dat correspondiente a la red de modelación.

Web App Dashboard Base de datos Gestión Control de acceso
