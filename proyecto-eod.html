Análisis de bases de datos de EOD
Herramienta de análisis de bases de datos
Descripción

Sistema de procesamiento y visualización de datos de Encuesta Origen-Destino. Permite analizar estadística descriptiva de varias bases de datos de Encuestas Origen Destino de forma simultánea y expedita.

Permite cargar varias bases de datos para analizarlas en simultáneo.

Para realizar el análisis debes responder a la pregunta: ¿Qué quieres analizar?, mediante el llenado de los campos correspondientes.

Presenta un resumen de las bases de datos cargadas: cantidad de hogares, de personas y de viajes

Una vez terminados los análisis, pueden volver a consultarse, ordenados por comuna

Al finalizar la sesión, se puede exportar un excel con la información.
