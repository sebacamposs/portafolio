Jupiter
Generador de redes SATURN
Descripción

Jupiter es un generador de redes de modelación SATURN, a partir de información de Open Street Map

Se define un polígono sobre el mapa Se seleccionan las categorías de vías a incluir
Se definen velocidades y flujos de saturación
y se genera una red .dat a partir de esta información
